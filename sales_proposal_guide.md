## 1. 당사 솔루션의 기술적 특징 (고객사 설명용)

고객사 미팅 시 당사가 개발한 시스템의 안정성과 최첨단 디자인 기술을 설명하는 세 가지 요점입니다.

### ① 그누보드5 기반의 안정적인 데이터 연동
*   **구현 내용**: 국내에서 보안성과 안정성이 가장 검증된 그누보드5 프레임워크를 기반으로 코어 테마를 제작했습니다. 임시 정적 페이지가 아닌, 데이터베이스(MySQL)와 실시간 연동되는 상태로 게시판 및 회원 스킨을 구현했습니다.
*   **고객사 설명 방향**: "저희가 구축한 시스템은 표준 그누보드5 엔진과 유기적으로 결합되어 있어, 관리자 페이지를 통해 공지사항 작성이나 FAQ 관리, 치료 사례 업데이트를 외주 개발사 도움 없이 원내 행정 팀이 즉시 100% 자가 관리할 수 있습니다." *(※ 불필요한 환자 개별 문의 대응 리소스 및 보안 우려를 줄이기 위해 별도 문의 폼은 연동하지 않았습니다)*

### ② 초고속 화면 전환 및 엘리먼트 동적 렌더링
*   **구현 내용**: 무거운 효과 플러그인을 배제하고, 브라우저 자체의 효율적 감시 기술인 Intersection Observer를 사용하여 화면에 텍스트와 사진이 부드럽게 솟아오르는 리빌(Reveal) 모션을 구현했습니다.
*   **고객사 설명 방향**: "원페이지 및 서브페이지 로딩 속도를 극대화하기 위해 백그라운드 렌더링을 최적화했습니다. 스크롤 위치에 맞춰 화면 요소들이 부드럽게 나타나므로 방문자에게 세련된 느낌을 제공함과 동시에 웹 성능 부하를 줄였습니다."

### ③ 고성능 스무스 스크롤 및 글래스모피즘 접이식 멀티 퀵메뉴
*   **구현 내용**: 부드러운 화면 흐름을 유도하는 Lenis 스크롤 라이브러리를 바인딩하였으며, 투명하고 입체감 있는 유리가 얹어진 듯한 **글래스모피즘(Glassmorphism) 스타일의 초정밀 퀵메뉴**와 **접기/펼치기(+) 마이크로 모션**을 구축했습니다.
*   **고객사 설명 방향**: "화면 스크롤 상태와 무관하게 항상 접근이 가능한 고정 퀵메뉴입니다. 병원 전반의 고급 테마 톤앤매너에 맞추어 인위적인 원색 버튼을 배제하고, 세련된 반투명 세이지 그린 라인과 인라인 벡터 SVG 아이콘으로 리디자인했습니다. '전화 문의', '카톡 상담', '맨 위로(Top)' 기능을 집약하되, 모바일 기기에서의 시인성을 해치지 않도록 **접기/펼치기(+) 단추**를 적용해 누르면 부드러운 회전 전환과 함께 스르륵 접혀 정돈되는 프리미엄 UI 인터랙션을 제공합니다."

## 5. 외부 플랫폼 연동을 위한 실제 주소(링크 URL) 수급 필요 고지

사이트 내에 장착된 길찾기 및 네이버 예약, 카카오톡 상담 등의 외부 서비스들이 정상적으로 작동하려면 고객사(병원) 측에서 개설하신 실제 외부 링크 URL 정보가 수급되어야 합니다.

*   **수급이 필요한 외부 링크 리스트**:
    1.  **네이버 길찾기**: 네이버 지도 등록 후 생성되는 '삼성더클성장의원' 길찾기 목적지 단축 URL
    2.  **카카오 길찾기**: 카카오맵 등록 후 생성되는 길찾기 목적지 고유 링크 URL
    3.  **티맵 길찾기**: TMAP 지도 정보 상의 고유 목적지 공유 링크 URL
    4.  **카톡 상담**: 병원 전용으로 개설한 카카오톡 채널(구 플러스친구) 채팅방 연결 주소 (예: `pf.kakao.com/...`)
    5.  **네이버 예약**: 네이버 예약 서비스 신청 완료 후 발급되는 예약 캘린더 바로가기 URL
